MAX & PRIMROSE — THE STORY OF US (forever version, GitHub Pages)
=================================================================
This folder is the permanent home of the site, with FULL-QUALITY videos.
Videos autoplay (muted) when scrolled into view — tap a video to toggle sound.
Fonts are embedded in the file: no external requests, works offline-forever.

DEPLOY (one time, ~10 min):
1. GitHub -> New repository (e.g. "mp-story"). Keep it PUBLIC (required for free Pages);
   the URL itself stays obscure and the page tells search engines not to index it.
2. Upload ALL contents of this folder (index.html + img/ + media/).
3. Settings -> Pages -> Deploy from a branch -> main, / (root) -> Save.
4. Wait ~1 minute. Forever URL: https://<username>.github.io/mp-story/
5. Send that URL back to Claude to generate the QR + the printed card.

ADD A MEMORY LATER (the living archive):
- Easiest: bring new photos + a few words to Claude -> get an updated index.html
  -> replace the file in the repo. Same URL, the printed QR never changes.
- Manual: drop photos into img/, duplicate one <article class="moment"> block in
  index.html and edit date/title/text.
